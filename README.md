# Binary Classification: Predicting Diabetes Using Neural Network
Steps to run the code.

1. Download “Diabetes_Prediction_Neural_Network.ipynb” or “Diabetes_Prediction_Neural_Network.py” file and “diabetes.csv”.
2. Open google colab (If prompted, use gmail account to login), upload "Diabetes_Prediction_Neural_Network.ipynb" (File -> Upload Notebook -> Choose file) or follow the link: https://colab.research.google.com/drive/13Nl0X0okm1azCS5W-LDUthbybObyA6Dn#scrollTo=L8ANT7_ZJA-u
3. Upload “diabetes.csv” under the files section (left sidebar with folder shape icon: File -> Upload to session storage (upload icon)).
4. Run the code to get the result.
5. Refer to the project documentation for complete details: "Diabetes_Prediction_NN_Documentation".
